# dotfiles

## Tools
+ `i3` as a window management
+ `polybar` for top bar
+ `urxvt` (rxvt-unicode) for terminal emulator
+ `compton` for enabling transparency with urxvt
+ `ranger` as a file manager
+ `neovim` for text editing
